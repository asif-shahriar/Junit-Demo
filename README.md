# Junit-Demo
## Prerequisites
* Install jdk 1.8 or any LTS version
* Configure **JAVA_HOME** and **GRADLE_HOME**
* Stable internet connection

## How to run this project
* Clone the repo
* Open command in the root folder
* Give the following command:
```
gradle clean test
```
